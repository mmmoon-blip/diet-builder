## 1. 快速开始

### 1.1 登录方式

**方式一：游客模式**
- 点击"游客模式"即可快速体验全部功能
- 系统会自动生成设备ID

**方式二：微信登录**
- 点击"微信一键登录"
- 使用微信授权登录

**方式三：手机号登录**
- 输入手机号
- 获取并填写验证码
- 完成登录

### 1.2 完成建档

首次使用请完成建档，这有助于AI为你提供更精准的服务：

```
我叫[昵称]，今年[年龄]岁，是[男/女]，身高[身高]cm，
初始体重[体重]公斤，目标体重[体重]公斤，
我是[减重/塑形/维持/增肌]
```

也可以分多次告诉AI，比如：
- "我叫小明"
- "我今年25岁"
- "我是女生，身高165cm，体重70公斤"

## 2. 首页功能

### 2.1 体重趋势图

在首页顶部展示您的体重变化趋势：

- **近7天 / 近30天 / 近90天**：点击切换时间范围
- **图例说明**：左上角显示绿色点=最低、橙色点=今日、红色点=最高
- **绿线**：目标体重线（当设置目标体重时显示）
- **数据点标签**：在数据点上显示体重数值

**统计摘要**：
- 累计变化：相比记录开始时的体重变化
- 最新体重：该时间段内最新的体重记录
- 距目标：距离目标体重还剩多少

### 2.2 身体维度图

展示身体各部位围度的变化趋势：

- **腰围 / 臀围 / 体重**：点击切换查看不同维度
- **图例说明**：绿色点=最低、红色点=最高
- **统计卡片**：显示当前值、累计变化、平均值

### 2.3 功能入口

## 3. AI助手

### 3.1 对话方式

在AI助手页面，您可以：

- **打字输入**：直接发送消息
- **快捷短语**：点击预设的快捷回复

### 3.2 支持的对话

**获取每日方案**
```
发送："今日方案" 或 "今天怎么吃" 或 "今天做什么运动"
AI会回复：包含今日饮食建议、运动建议、小贴士的完整方案
```

**记录体重**
```
发送："今天体重70公斤"
AI会自动记录到您的档案中
```

**打卡签到**
```
发送："打卡" 或 "签到"
AI会记录您的打卡，并给出鼓励
```

**记录站立**
```
发送："站起来了" 或 "站立"
AI会记录您的站立活动
```

**情绪倾诉**
```
发送："心情不好" / "暴食了" / "焦虑" / "沮丧"
AI会主动安慰您，并给出建议
```

**日常对话**
```
发送：任何关于减肥、饮食、运动的问题
AI会结合您的个人情况给出建议
```

### 3.3 AI自动识别

AI会自动识别您的消息意图，无需特定格式：

- 检测到体重数字 → 自动记录体重
- 检测到负面情绪 → 主动情绪安抚
- 检测到"方案"、"计划" → 生成每日方案
- 检测到"打卡" → 记录打卡

## 4. 体重记录

### 4.1 快速记录

1. 点击首页的 ⚖️ 图标
2. 输入今日体重
3. 可选：记录睡眠时间
4. 点击"记录体重"

### 4.2 导入历史记录

如果您之前有在其他地方记录体重，可以批量导入：

1. 点击"导入历史记录"
2. 填写：
   - 减肥开始日期
   - 初始体重
   - 目标体重
3. 点击"生成日期"
4. 系统会自动生成日期列表
5. 修改每天的体重数据
6. 点击确认导入

### 4.3 查看历史

- 向下滚动查看更多历史记录
- 点击 ✏️ 编辑记录
- 点击 🗑️ 删除记录

## 6. 经期记录

### 6.1 记录经期

1. 点击首页的 🩸 图标
2. 在日历上选择经期开始日期
3. 可选：选择经期结束日期
4. 填写经量（少/中/多）
5. 选择是否有疼痛
6. 添加备注（可选）
7. 点击"记录经期"

### 6.2 查看周期

记录经期后，系统会自动：

- 显示当前处于哪个阶段
- 预测下次经期时间
- 给出当前阶段的运动、饮食建议

**经期阶段说明**：
- 📍 经期：1-7天
- 📍 卵泡期：第8-14天（减脂黄金期）
- 📍 排卵期：第15天左右
- 📍 黄体期：第16-28天

## 9. 体脂估算

### 9.1 计算方法

体脂率通过以下数据估算：

- 腰围
- 臀围
- 体重
- 性别
- 年龄

### 9.2 查看结果

计算后会显示：

- **体脂率**：估算的体脂百分比
- **体脂类别**：
  - 偏低（运动员水平）
  - 标准（健康范围）
  - 偏高
  - 过高
- **AI建议**：根据体脂情况给出的建议

## 10. 常见问题

### Q1: 忘记记录体重怎么办？

可以补记过去的体重：
1. 打开体重记录弹窗
2. 点击日期选择器
3. 选择需要记录的日期
4. 输入当时的体重

### Q2: 为什么体重突然变重了？

可能的原因：
- 经期前水肿（正常现象）
- 昨晚吃太咸
- 昨晚碳水摄入较多
- 运动后肌肉增加

建议：关注围度变化而非只看体重

### Q3: 平台期怎么办？

平台期（体重停滞2-4周）是正常现象，尝试：
- 增加力量训练提高基础代谢
- 换一种运动方式给身体新刺激
- 保证7-9小时睡眠
- 不要过度节食

### Q4: 暴食了怎么办？

1. 不要自责，这是很常见的
2. 明天正常吃就好，不要断食
3. 多喝水帮助代谢
4. 适当增加运动
5. 2-3天后体重会恢复

### Q5: 如何联系客服？

如有问题可以在AI助手页面询问，AI会尽力帮助您解答。

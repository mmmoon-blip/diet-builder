# 减减 - 登录模块配置说明

## 登录方式

### 1. 小程序端（微信开发者工具）
- **手机号登录** - 输入手机号，获取短信验证码登录
- **微信一键登录** - 使用微信小程序的 wx.login() 获取 openid 登录
- **游客模式** - 无需注册，直接体验

### 2. Web端
- **游客模式** - 仅支持游客体验完整功能

---

## 后端配置

### 数据库
无需额外建表，使用 Hibernate 的 `ddl-auto: update` 自动创建/更新表结构。

### 配置文件
`server/src/main/resources/application.yml`

```yaml
# JWT 配置（Token加密用）
jwt:
  secret: diet-butler-secret-key-32-chars-minimum-for-hs256-algorithm
  expire-days: 7

# 微信小程序配置（用于微信登录）
wechat:
  appid: your-wechat-appid           # 替换为你的小程序appid
  secret: your-wechat-secret         # 替换为你的小程序secret

# 短信服务配置（用于手机号登录）
sms:
  enabled: false                      # 开发环境设为false
  aliyun:
    access-key: your-aliyun-access-key
    secret-key: your-aliyun-secret-key
    sign-name: 减减
    template-code: your-template-code  # 短信模板CODE
```

---

## 短信服务配置（生产环境）

### 阿里云短信服务

1. 注册阿里云账号并开通短信服务
2. 创建短信签名和模板（模板需要包含变量 `{code}`）
3. 获取 AccessKey ID 和 AccessKey Secret
4. 将上述配置填入 `application.yml`

示例模板：
```
您的验证码是${code}，5分钟内有效。
```

## API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/auth/sendSms` | POST | 发送短信验证码 |
| `/api/auth/phoneLogin` | POST | 手机号登录 |
| `/api/auth/login` | POST | 微信登录 |
| `/api/auth/guest` | POST | 游客登录 |
| `/api/auth/me` | GET | 获取当前用户信息 |
| `/api/auth/validate` | POST | 验证Token |
| `/api/auth/logout` | POST | 登出 |

## 微信小程序配置

### 1. 微信开发者工具
在 `diet-butler-mini` 项目中：

1. 打开 `project.config.json`，确认 `appid` 正确
2. 确保不勾选"不校验合法域名"（开发阶段可勾选）
3. 后端地址：`http://localhost:8080`

### 2. 小程序登录流程
```javascript
// 1. 获取手机号（用户授权）
wx.getPhoneNumber({
  success: (res) => {
    // 发送到后端验证
  }
})

// 2. 微信登录（获取openid）
wx.login({
  success: (res) => {
    // 用code换取openid
  }
})
```

### 3. 游客模式
自动生成设备ID，无需用户操作。

## 开发调试

### 启动后端
```bash
cd server
JAVA_HOME="/Library/Java/JavaVirtualMachines/jdk-21.jdk/Contents/Home" mvn compile
JAVA_HOME="/Library/Java/JavaVirtualMachines/jdk-21.jdk/Contents/Home" java -cp "target/classes:$(mvn dependency:build-classpath -q -DincludeScope=runtime -Dmdep.outputFile=/dev/stdout)" com.dietbutler.DietButlerApplication
```

### 启动小程序
1. 打开微信开发者工具
2. 导入 `diet-butler-mini` 项目
3. 确保后端运行在 `localhost:8080`
4. 点击"编译运行"

### 测试账号
- 手机号登录：任意有效格式手机号，开发模式验证码在控制台打印
- 微信登录：需配置真实的微信小程序 appid 和 secret
- 游客登录：直接使用

## 注意事项

1. **生产环境**务必设置：
   - `jwt.secret` 为复杂随机字符串
   - `sms.enabled: true` 并配置阿里云短信
   - `wechat.appid` 和 `wechat.secret` 为真实小程序配置

2. **Token有效期**默认7天，可通过 `jwt.expire-days` 修改

3. **Web端**微信登录不可用（需微信开放平台资质），建议引导用户使用小程序
